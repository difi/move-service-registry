[#fiks-api]
= Manage FIKS protocol mappings

|===
|HTTP Method |URL |Description |Media type(s)

|POST
|<<fiks-post,/api/v1/fiks>>
|Add/update mapping
|application/json

|DELETE
|<<fiks-delete,/api/v1/fiks/{identifier}>>
|Delete mapping
|application/json

|GET
|<<fiks-get,/api/v1/fiks>>
|List mappings
|application/json

|GET
|<<fiks-findby,/api/v1/fiks/{identifier}>>
|Find by identifier
|application/json

|===
